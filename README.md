## Intro
An infinite scroll plugin for Vue 3, to help you implement an infinite scroll list more easily.

## Usage

```sh
npm i @codog/vue3-infinite-loading
```

### Features
- Mobile friendly
- Internal spinners
- 2-directional support
- Load result message display
